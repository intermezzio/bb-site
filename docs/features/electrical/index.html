How BB-8 is powered and controlled

An Arduino controls all of the electrical components on BB-8. The Arduino is connected to various other components, including motors and a bluetooth module, to drive BB-8.

Bluetooth

BB-8 uses a HC-05 Bluetooth chip to communicate between the Arduino and a laptop.
